Formulario - HTML

Os formulários, são a inserção de dados através de um usuário, imagine que o mesmo precise inserir um login e uma senha, a tag formulário abrirá tal opção. Na abertura da tag, é preciso inserir dois módulos, o action (para onde os dados serão enviados) e o módulo method (para a postagem dos dados).

    <form action="site" method="post">
  • texto

    • Onde será registrado textos acionados pelo usuário, como logins e coisas parecidas, que necessitam do tipo string. Sendo necessário os módulos type, name e placeholder:

            <input type="text" name="texto1" placeholder="Insira seu texto aqui">
  • Senhas

    • Possuem a mesma lógica do texto, porém, o que será escrito estará ocultado. Para isso, será preciso os módulos type e name:

            <input type="password" name="senha1"
  • Opções de escolha

    • Com esta tag, será aberto uma caixa de opções, onde o usuário deverá optar por uma. Sendo que as opções listadas serão elaboradas pelo programador. Para isso, é preciso abrir a tag select com o módulo name e option, as quais receberão os valores de opções da caixa:

            <select name="seleção">
            <option value=1> opção <1option>
        </select>
  • radio

    • Os inputs do tipo radio, exibem opções de marcação para o usuário, de forma que o mesmo só poderá marcar uma opção. Para este feito, é preciso da tag input com os módulos type, name e value. Saiba que todas as opções recebem o mesmo nome, porém com valores distintos:

            <input type"radio" name="radio1" value="Nº"> opção
  • CheckBox

    • O checkBox possui a mesma sintaxe lógica do radio, com opções de marcação, com a diferença que o usuário podrá selecionar mais de um>

            <input type"checkbox" name="ck1" value="Nº"> opção
  • submit

    • Este input , enviará todos os dados registrados no site ou local a ser recebido. Sendo preciso a tag input e os módulos type, value e name:

            <input type"submit" value="enviar" name="submit">

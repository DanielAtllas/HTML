Tags - HTML

As tags, são funções dentro do HTML que possuem alguma fomração no texto inserido, podendo tornar títulos, parágrafos, quebras de linhas e muitas outras coisas. Basicamente, as tags fazem parte de 99% dos scripts em HTML. Existem inpumeras tags de HTML, dessa forma, é aconselhável pesquisar na internet sobre a ação que se deseja realizar. Seguem as principais tags e mais usadas em um script HTML:

  • Título

    • Há uma forma muito simples de acrescentar títulos nos scripts dos programas, para isso, é usada a tag h do Nº1 até o Nº6, de forma que se diferem pelo tamanho da fonte e hierárquia (Logo, uma tag h2 está dentro da tag h1 e assim por diante). É aconselhável não usar as tags apenas pelo tamanho, mas sim como sequência de títulos. Pois isso não dará nexo na leitura do script.

                <h1> Título 1 </h1>
            <h2> Título 2 </h2>
  • Parágrafos

    • Como um script HTML é basicamente a inserção de um texto, são necessários parágrafos neste texto para dar nexo ao mesmo, dessa forma, é usada a tag p, pois se, essa tag, todo o texto será exibido em uma linha unitárias.

    Diferente da lingua portuguesa, no HTML, o parágrafo significa que informações estão interligadas. Além disso, para conhecimento, é boa a compreenção de que caracteres especiais estão restritos a linguagem, ou seja, um problema recorrente é na hora de inserir o sinal de menor o qual é o mesmo sinal usado para abertura de uma tag. Portanto, ao inseri-lo, o script o entenderá como uma tag, precisando assim de um código para sua apresentação. Segue um link com todos os caracteres especiais e o código para serem acionados:

    Caracteres Especiais 
                <p> Parágrafo </p>
  • Texto cru

    • Há também o uso da tag pre, a qual irá exibir o texto inserido da forma que foi excrito. Porém, essa tag é pouco usada, por mais prática que seja. Um exemplo de seu uso, são os exemplos acima apresentaos (perceb a diferença no texto).

                <pre> texto </pre>
  • Quebra de linha

    • É muito usual a quebra de linhas nos textos, de forma que as linhas apresentem um mesmo objetivo, mas em linhas diferentes. Para isso, é usada a tag br. Um bom exemplo de seu uso, é ao inserir ordenações, em que os textos ficam muito juntos uns dos outros. Nesses casos, o uso da quebra de linha é ótimo para ajustar e melhorar a parte visual gerada.

                <p> 
                Informação 1
                <br>
                Informações 2
            </p>

    Perceba que essa tag em especial, nãoo exige uma 2 parte de fechamento.

  • Links

    • Algo presente em sites, é o uso de jumpers de links, de forma que ao clicar no jumper, o usuário é automaticamente redirecionado para o link. Para fazer isso no HTML, é usado a tag a, onde é preciso informar o link e um jumper.

                <a href="link">
                jumper
            </a>
  • Hiperlinks

    • Algo muito interessante a ser trabalhado no HTML, é o uso de hiperlinks com arquivos locais, de forma que a serem "chamados" pelo site. Imagine um site que possua um jumper para uma foto presente em seu computador. É pra isso que os hiperlinks servem. Dessa forma, qualquer arquivo pode ser passado, foto, pastas, arquivos, documentos e etc.

                <a href="diretório">
                jumper
            </a>
  • Listas ordenadas e não ordenadas

    • Outra função presente no HTML com tags, é poder listar um conteúdo de forma ordenada ou não. Para isso, são usadas 2 tags, a ul ou ol (para iniciar a ordenação) e a tag li para a sequência da lista. Onde ul é para listas não ordenadas e a tag ol é para listas ordenadas (Também é possível inserir listas dentro de outra lista, independente de sua ordenação).

        <h1> Experiências </h1>
	 <ul>
		<li> Atual </l1>
			<ul>
				<li> Coodenador </l1>
				<l1> Professor </l1>
			</ul>

		<l1 Anterior </l1>
			<ol>
				<l1> Analista </l1>
				<l1> DBA </l1>
			</ol>
	</ul>
  • Inserção de imagens

    • Algo presente em sites, são as imagens e por mais que o CSS seja responsável pela parte visual do site, o HTML deve apresentar quais e as posições em que as imagens serão postas. Usa-se o conceito semelhante a hiperlinks em tal função dentro do parágrafo. Além de ser apresentado o endereço, tamanho e a descrição da imagem. Saiba que essas imagens não precisam ser locais, tendo apenas que inserir o link da imagem caso ela se encontre na rede.

                <p>
                <img src="local da imagem width="largura" height="altura" alt="Descrição">
            </p>

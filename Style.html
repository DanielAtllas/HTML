Style - HTML

É possível inserir cores nas fontes dos textos do HTML, assim como muitas outras variações em suas letra, palavras ou parágrafos, onde para isso, é necessário o uso da tag style no head do script, além de informar quais tags irão aderir a tal definição. Segue abaixo uma lista das principais usadas (mas lembre-se que existem inúmeras variações. Pesquise-as para mais informações):

  • Coloração

    • A inserção de cores irá alterar a colocarção padrão do texto, onde deverá ser passada as informações necessárias para isso, como o que será mudado e a cor que se deseja. Em relação a cor desejada, inpumeras paletas de cores existem e os códigos específicos para cada uma, no link abaixo, está presente uma tebela destas:

    Cores 
                <head>
                <style>
                    p{color:cor}
                </style>
            </head>
            
            Perceba que a tag p foi passada, dessa forma, todos os textos que estiverem dentro da tag
            serão exibidor da cor definida. Mas caso seja de opção apenas um parágrado, é dado o comando dentro da tag 
            de parágrafo:
            
            <p> style="color:cor" texto </p>

    Nesta mesma definição, é possível inserir inúmeras outras definições, como blink e até mesmo a fonte do texto.

    Tabelas

    Para a criação de yabelas no HTML, é preciso abrir a tag table e inserir o título as célular com a tag td. É preciso também, definir o tamanho da borda, largura e cor, como no exemplo abaixo:

                <h3> titulo </h3>
            <table border=>Nº width="largura">
            <tr colspan="4" bgcolor="cor"
                <td> coluna 1 </td>
            </table>

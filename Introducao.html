Introdução - HTML

O HTML é um tipo de linguagem de programação para WEB, a qual é vinculada a navegadores web criando páginas interativas ou não. Basicamente, cada linguagem WEB, entre as 3 principais, possui uma função distinta, sendo eles (de forma bruta):

  • CSS - Responsável pela parte "visual".

  • HTML - Responsável pelo "texto e algumas imagens".

  • Java Script - Responsável pela parte "interativa".

Neste caso, o HTML sendo baseado em "somente texto", podendo ser feito em qualquer editor de texto (como bloco de notas) desde que o mesmo seja salvo em formato HTML. Ou seja, independendte de onde se crie o script, se o mesmo for salvo em formato HTML, qualquer sistema operacional irá entende-lo como tal. Dessa forma, qualquer texto escrito, será gerado como programa em um navegador. Esses textos possuem as chamadas tags para cada função na linguagem. São essas tags que definem o que cada parágrafo u extensão dele, fará. Desde criação de um parágrafo de texto, inserção de imagens, coloração e intonação de palavras e etc. Essas tags são indicadas pela abertura e fechamento do sinal de maior/menor, inserindo a tag desejada dentro deste comando. Como no exemplo a seguir:

            <tag>
            
            </tag>
  • Comentários

    • Como qualquer linguagem, é possível inserir comentários no script, de forma que o mesmo não irá interferir ou alteraro script, funcionando apenas como informação adicional ou explicativa do script ou parte dele. Para isso, é usada a tag:

                <!-- 
            
                Comentário
                
            -->
  • Script

    • O inicio do script HTML segue um padrão, sendo exigido um cabeçalho, o head e o body.

    1. Cabeçalho

      2. Será declarado o tipo do script (html no caso) e a abertura da tag html que dará inicio ao programa. Além disso, é no cabeçalho que será definido o tipo de linguagem que será usada na escritura do script (português e etc).

                          <!DOCTYPE html>
                    <html lang="pt-br">
                        script
                    </html>

    2. Head

      3. Área onde serão anexaddas as declarações, ou seja, são informações que orientam o navegador e o script. Portanto, o head é onde as declarações do script serão armazenadas, como por exemplo o tipo dos caracteres{ (<meta charset="utf-8">) (<meta name="description" content="descrição")} e o título da aba (<title> titulo </title>), essas são declarações obrigatórias. Porém, outras tags também podem ser contidas no head, como a descrição da página, chamador de arquivos ou scripts de outras linguagens, coloração de parágrafos e etc.

                          <head>
                        tags
                    </head>

    3. Body

      4. Composição e estrutura do script em sí, onde a interface e as tags serão incrementadas. Portanto, é no body que o script em sí ficará, ou seja, o que será "viso" pelo usuário como face gráfica do programa. É aqui onde as principais tagas serão usadas, como os seguintes exemplos:

                          <h1> = Título;
                    
                    <p> = Parágrafos;
                    
                    <strong> = negrito;

    Portanto, de forma resumida, um script em HTML ficaria da seguinte forma:

                        <!DOCTYPE html>
                    <html lang="pt-br">
                        <head>
                            <meta charset="utf-8">
                            <title>Menu HTML</title>
                            <meta name="description" content="Este é um site referênte à estudos de HTML">
                        </head>
    
                        <body>
                            SCRIPT
                        </body>
                    </html>
